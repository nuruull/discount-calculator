# Python Project — Discount Calculator GUI

## Project Overview

This project is created as part of the Advanced Teacher Project preliminary stage.

The goal of this project is to build a simple Python application that calculates the total price after applying a discount. The program uses a Graphical User Interface (GUI) so users can interact with the calculator more easily.

This project is designed to demonstrate basic Python programming skills, including:
- User input handling
- Mathematical calculations
- Error handling
- GUI development using Tkinter

---

## Features

The application allows users to:

- Enter product price
- Enter discount percentage
- Enter quantity of items
- Calculate total price before discount
- Display discount amount
- Show final price after discount

---

## Technologies Used

- Python
- Tkinter (Python GUI Library)
- Visual Studio Code (VS Code)

---

## How to Run the Project

### Step 1 — Open Project in VS Code

Open the project folder in Visual Studio Code.

### Step 2 — Run Python File

Run the file:

```python
discount_calculator.py
```

or use terminal command:

```python
python discount_calculator.py
```
### Step 3 — Use the Application

1. Input product price
2. Input discount percentage
3. Input quantity
4. Click the Calculate button
5. View the final result

## Project Purpose

This project helps students understand:
- Basic Python syntax
- Variables and data types
- Functions
- Simple business calculations
- GUI application development

It is also suitable for beginner learners aged 8–16 years old because the logic is simple and easy to understand.
